init(privateKey, issuerEmail) → {Chatv1}

Create an object that can build requests and parry responses back via the Chatv1 API. It uses Oauth2 library to authenticate, and thus privateKey and issuerEmail is required. Continue interacting with api via the Chatv1 object it returns.

Parameters:
Name Type Description
privateKey string

Private Key as obtained via service account
issuerEmail string

Issuer Email as obtained via service account

Example
const Chat = ChatService.init('<privatekey>', '<issueremail>');
const response = Chat.Spaces.list();
